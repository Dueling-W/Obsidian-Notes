## GitHub for all Obsidian Notes - self learning and computers science courses


### Folder Structure

- 00 - Rough Notes: reserved for rough notes and brainstorming, can include general plans for the week
- 01 - Source Material: direct notes from live lectures, videos, books, etc.
- 02 - Tags: a collection of tags for all the various notes (i.e., computer vision, NLP, etc.), should be moderately specific
- 03 - Templates: a small folder of templates for new notes
- 04 - Full Notes: complete notes from the source material; will be focused on a specific area/topic, such as Random Forest classifiers
- 05 - References: general definitions, formula collection, cheat sheets; ex: a cheat sheet on important integrals or linear algebra topics
